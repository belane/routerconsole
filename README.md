## Console for 3com 812 and SpeedStream 5660

Remote management console for 3com 812 and SpeedStream 5660 routers.

### Features

* Router Status
* Router config
* Conections
* Reboot
* Show NAT usage
* Show NAT servers
* Reset NAT table
* Enable / Disable Firewall
* Show filters
* Open internal servers ports
* Configurable servers
* Systray menu
